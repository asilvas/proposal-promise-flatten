Introduction

Background for tc39/proposal-safeawait, inspired by David Wells. Please report any issues.

safeawait _promise_

Built atop `await`, `safeawait` simply returns all outcomes of a Promise into a tuple `[rejectValue, resolveValue]`:

1. Always return a tuple. 1. Promise `reject` returned as-is at index zero of the returned tuple, or `undefined`. 1. Promise `resolve` returned as-is at index one of the returned tuple, or `undefined`. 1. Never throws.
